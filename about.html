Sobre esta aplicação


  • Os dados apresentados através dos gráficos são atualizados diariamente de acordo com a disponibilidade dos serviços que fornecem os dados referentes à contaminação por Covid-19.
  • Os gráficos são interativos. Para visualizar melhor as estatísticas, clique/toque sobre os elementos do gráfico para os dados sejam exibidos.
  • Se estiver utilizando dispositivo móvel, utilize a orientação de paisagem (horizontal) para visualizar e interagir melhor com os gráficos.
  • Para maiores informações, envie um e-mail .
